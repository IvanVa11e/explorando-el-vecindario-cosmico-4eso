Cómo empezar

Una vez tengáis asignado el planeta sobre el que vais a trabajar, utilizaremos las siguientes herramientas para llevar a cabo la misión:

  • Solar System Scope: vuestro centro de mando para investigar datos fiables en tiempo real.
  • Canva: para dar vida a vuestras infografías interactivas.
  • Herramientas de audio: para grabar el podcast que nos llevará a otros mundos.
